## –ö–æ–º–ø–æ–∑–∏—Ç–Ω—ã–π –∏–Ω–¥–µ–∫—Å
**Urban Quality Index ($UQI$)** - *–∫–æ–º–ø–æ–∑–∏—Ç–Ω—ã–π* –∏–Ω–¥–µ–∫—Å –∫–∞—á–µ—Å—Ç–≤–∞ –≥–æ—Ä–æ–¥—Å–∫–æ–π —Å—Ä–µ–¥—ã, –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç—å—é –∫–æ—Ç–æ—Ä–æ–≥–æ —è–≤–ª—è–µ—Ç—Å—è:

* –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –æ—Ç–∫—Ä—ã—Ç—ã—Ö –¥–∞–Ω–Ω—ã—Ö

$UQI$ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π —Å—É–º–º—É –¥–≤—É—Ö –∏–Ω–¥–∏–∫–∞—Ç–æ—Ä–æ–≤: **–æ–±—ä–µ–∫—Ç–∏–≤–Ω–æ–≥–æ** –∏ **—Å—É–±—ä–µ–∫—Ç–∏–≤–Ω–æ–≥–æ**:
